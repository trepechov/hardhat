# Hardhat demo

This project contains a couple of solidity contracts and corresponding unit tests.

* Token.sol - Hardhat demo contract
* USElections.sol
* TechnoStore.sol

## Installation

Use npm to install project dependencies

```bash
npm install
```

## Usage

```python
npx hardhat compile
```

## Testing


```bash
npx hardhat test
```
